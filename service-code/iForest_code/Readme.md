# 算法复现与对比实验项目

本项目主要包含孤立森林（Isolation Forest, iForest）算法的复现源码，以及对比实验报告和结果可视化。

## 📁 目录结构说明

根据项目根目录，各文件夹及文件的功能定义如下：

* **`iforest_code/`**
    * **用途**：核心算法实现目录。
    * **内容**：包含孤立森林（Isolation Forest）算法的复现源码、核心模型或算法逻辑的具体实现。
* **`model1report/` & `model2report/`**
    * **用途**：对比模型实验报告/记录。
* **`raw_data/`**
    * **用途**：原始数据集目录。
    * **内容**：存放未经处理、最纯粹的源数据输入
* **`result_figure/`**
    * **用途**：实验结果可视化。
    * **内容**：存放模型运行后生成的图表（如 ROC 曲线、PR 曲线、异常检测得分散点图、评估指标对比图等可视化图片）。
* **`Readme.md`**
    * 项目的引导说明文档（即当前文件）。
